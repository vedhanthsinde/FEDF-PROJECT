# 🏆 PlaySpot

PlaySpot is a sports venue booking platform developed using React.js. The application helps users discover nearby sports facilities, check availability, and book slots quickly and easily.

## 📌 Project Overview

Students and working professionals often face difficulties finding sports grounds and enough players for games. PlaySpot solves this problem by providing a simple platform where users can:

- Find nearby sports venues
- Browse available sports facilities
- Book slots instantly
- Explore different sports categories
- Contact the platform through a built-in contact form

## 🚀 Features

- User Login System
- User Registration System
- Home Page Dashboard
- Sports Venue Listing
- Category-Based Filtering
- Booking Simulation

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/vedhanthsinde/FEDF-PROJECT.git
```

### Navigate to Project Folder

```bash
cd FEDF-PROJECT
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open Browser

```
http://localhost:5173
```

## 📖 How It Works

1. User logs in or signs up.
2. User navigates to the Spots page.
3. User selects a sport category.
4. Available venues are displayed.
5. User books a slot (demo implementation).
6. User can contact the platform through the contact form.
